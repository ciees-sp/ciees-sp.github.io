Picker layout mockups

Two options for adding character thumbnails to the audience-track picker. Both inside the existing white frame card. The card morphs into the scenario card on selection.

Layout A

Stacked cards with top image strip

Same vertical flow as the current engine. Each card grows: full-width photo on top, then name and description below. Three cards stack inside the frame. Closest to the existing engine pattern; clean responsive behavior; no breakpoint surgery.

Layout B

3-column gallery

Three cards side-by-side at desktop, each with a 4:3 photo on top and shorter text below. Frame widens to 920px at desktop to give the columns breathing room. Stacks to a single column on mobile (≤720px). More gallery-feel; lets the three populations be compared at a glance.
